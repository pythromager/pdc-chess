📜 PDC Rules

🎯 Objective

Checkmate the opponent or put them in an impossible must situation.

♙ Pawn

  • Can only capture other pawns
  • Can also capture knights in the river (blue squares)

♖ Rook

  • MUST capture if any piece is within 3 squares
  • Can only capture pieces within 3 squares

♘ Knight

  • Can only capture pieces that are currently giving check

♗ Bishop

  • MUST give check if it can move to a checking square
  • Cannot capture any pieces

♕ Queen

  • MUST capture if a non-knight is in range
  • Cannot capture knights

♔ King

  • MUST move when in check (cannot block or capture)
  • Cannot capture while in check

🌊 The River (c4-f5)

The blue highlighted squares. Pawns can capture knights here.

⚡ Must Override

You can ignore a must ONLY if you deliver checkmate in 1 move.

📊 15-Move Space Rule

If 15 moves pass without a capture, the player controlling more space wins.

🏆 Win Conditions

  • Checkmate
  • Opponent has 2+ conflicting musts
  • Opponent in check + another must
  • Time out
  • Resignation
